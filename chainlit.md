# Text-Image RAG

Upload a document and ask questions about it in any language.

**Supported formats:** `.pdf` · `.docx` · `.txt` · `.xlsx` · `.csv`

**Features:**
- Multimodal retrieval — text, tables, and images
- Hybrid search (vector + BM25) with cross-encoder reranking
- HyDE query enhancement for better retrieval accuracy
- 50+ languages supported natively
